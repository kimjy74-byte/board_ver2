=========================================
주간행사 일정 관리 위젯 (Weekly Event Schedule Widget)
=========================================

[사용 방법]
1. 단독 위젯/웹앱 실행:
   - '실행하기.bat' 또는 'index.html'을 더블클릭하시면 바로 브라우저에서 사용할 수 있습니다.
   - 브라우저 상단의 '위젯 모드' 버튼을 누르면 바탕화면 미니 뷰 위젯으로 전환됩니다.

2. 네트워크/모바일 연동 서버 실행 (선택 사항):
   - '서버실행_Python.bat'을 실행하면 로컬 웹 서버가 동작하며 QR 코드로 모바일 접속이 가능합니다.

[주요 기능]
- 주간 행사 일정 등록, 수정, 삭제 및 자동 완성
- 행사 완료 체크박스 및 메모 기능
- 엑셀(CSV) 데이터 내보내기 & 불러오기 백업 기능
- 일정 이미지(스마트폰 캡처본 등) Drag & Drop OCR/AI 자동 파싱
- 위젯 모드 & 대시보드 모드 자유 전환
- 작성 데이터 브라우저 로컬 스토리지 자동 저장
